# Pack De Ressources Francaises

Ce dossier contient des termes sources en francais et des suggestions de remplacement en francais.

Exemples :
* `maitre` -> `principal`, `primaire`
* `esclave` -> `secondaire`, `replique`
* `liste noire` -> `liste de blocage`, `liste d'interdiction`
* `liste blanche` -> `liste d'autorisation`, `liste de validation`
* `homme-heure` -> `personne-heure`
* `il`, `elle`, `il/elle` -> `iel`

Note sur `iel` :
* `iel` est utilise dans certains contextes d'ecriture inclusive, mais il n'est pas accepte partout.
* Utilisez cette suggestion selon votre public, votre guide de style institutionnel et le contexte de communication.

Validation :
* `python -m unittest tests.resource_schema_test`
* `python -m tests.tests`
